# Telegram-бот бот с ИИ-персонажем

[![eng](https://img.shields.io/badge/lang-en-en?color=DCDCDC)](https://github.com/allcodny/ai-character-telebot/blob/master/README.md)
[![rus](https://img.shields.io/badge/lang-ru-ru)](https://github.com/allcodny/ai-character-telebot/blob/master/README-ru.md)

Telegram-бот с интеграцией ИИ (через OpenRouter) для генерации текстовых ответов от лица любого персонажа.

## ✨ Возможности:
* Генерация осмысленного ответа в соответствии с характером персонажа, которого вы пропишите в промпте
* Распознание изображений
* Понимание контекста (личные сообщения, комментарий, чат, другая реакция на конкретных пользователей в чате)
* Настройка частоты/шанса ответа в разных контекстах
* Белые списки для безопасности и экономии запросов

## ⚙️ Установка и настройка

**Примечание:** Рекомендуется запускать бота на хостинге, любой телеграм бот требует постоянно запущенного кода для постоянной работы.

1. Предварительные требования
   - Python 3.8+
   - Токен вашего телеграм бота ([Где получить](https://t.me/BotFather))
   - Ключ [API OpenRouter](https://openrouter.ai/settings/keys) (Зарегистрироваться на сайте -> Create API Key -> Скопировать)

2. Клонирование репозитория
   <br>`git clone https://github.com/allcodny/ai-character-telebot`
   <br>*или*
   <br>скачать как ZIP файл и распаковать

3. Установка зависимостей
   <br>В командной строке: `pip install -r requirements.txt`

4. Настройка конфигурации
   <br>Измените содержание файла bot_config.py под ваши нужды в соответствии с комментариями в файле.
   <br>*Примечание: ID человека или группы можно узнать с помощью различных телеграмм ботов (например, Get Any ID)*

5. Создание промптов
   <br>Modify the contents of the bot_config.py file to suit your needs.
   <br>*Примечание: экспериментируйте с промптами! В главном промпте можете прописать всё, что считаете нужным. Рекомендуется в основном приписывать примеры сообщений персонажа.*

6. Запустите бота
   <br>`python bot.py`


## 📋 Использование
- Для личных сообщений: убедитесь, что вы находитесь в белом списке или белый список выключен. Напишите что-нибудь боту.
- Для комментариев/сообщений в чате: убедитесь, что чат находится в белом списке или белый список выключен. Убедитесь, что шанс ответа у бота проставлен верно. Напишите пост или сообщение в чате.


## О ИИ
В проекте используются бесплатные ИИ модели, которые лучше всего работали в **моём** случае. Вы можете выбрать любую другую ИИ модель на [OpenRouter](https://openrouter.ai/models) и изменить на её в конфиге


## ❗ Возможные проблемы:
- код 429 - ограничение [по запросам](https://openrouter.ai/docs/api-reference/limits). (Для бесплатных моделей это 20 запросов в минуту, 50 запросов в день)
- [Про другие ошибки](https://openrouter.ai/docs/api-reference/errors)
